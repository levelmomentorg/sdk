# Contributing

Thanks for your interest in the Level Moment SDK.

## Issues and pull requests

- **Bug reports and feature requests** — please
  [open an issue](https://github.com/levelmomentorg/sdk/issues). This is the best
  way to get a change made.
- **Pull requests** — feel free to open a PR to show exactly what you have in
  mind. The maintainers review incoming changes and fold accepted ones into an
  upcoming release; note that a PR may be incorporated as part of a release
  rather than merged directly, and you'll be credited in the changelog.

## Security

Please do **not** file security issues in the public tracker. Email
security@levelmoment.com instead.
